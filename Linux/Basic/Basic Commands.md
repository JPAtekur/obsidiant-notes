* Push directory
```
pushd /etc
```
* Back to push directory
```
popd
```

* Create parent directory
```
mkdir -p Asia/Bangladesh
```

* Write in to a file using cat command. Press Ctrl + d to save
```
cat > test.txt
```

* Show directories and files in accessing time order
```
ls -lt
```
reverse order
```
ls -ltr
```
